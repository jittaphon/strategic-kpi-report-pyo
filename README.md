## ğŸ¯ Project Objective

This project is a **web-based dashboard** designed to:
- Monitor provincial-level **Key Performance Indicators (KPIs)** aligned with the strategic plan of the Ministry of Public Health (Thailand).
- Support **data-driven planning and decision-making** at the local health administration level.

It consolidates health performance data from various reporting units and visualizes them for strategic management.
